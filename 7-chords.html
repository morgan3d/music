# ADGBEAD `A1–D2–G2–B2–E3–A3-D4`
This is close to standard A-Baritone tuning, but uses a B2 string instead of a C3 string. Play shapes as you would in standard tuning, but with the root on the 7th string instead of the 5th string (albeit tuned down an octave). Ignore the two highest pitched strings.
